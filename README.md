# Brief introduction
Water vapor feedback locking experiment is based on the idea that you feed the radiation module in the model code with constant year-to-year water vapor content, say from a control experiment. In other words, only modify the radiation effects of water vapor. You can choose to do either longwave part, shortwave part, or both.
# Getting started
You need to be able to run the CESM model. Check the radiation code used in the model. Normally the atmospheric component-CAM used RRTMG scheme.
